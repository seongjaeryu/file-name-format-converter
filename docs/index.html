파일명 포맷 컨버터

macOS 파일명(NFD)을 Windows 호환 포맷(NFC)으로 변환합니다

로컬 파일 직접 변환: 폴더 또는 파일을 드래그 & 드롭하거나 클릭하여 선택하세요.
원본 파일명이 바로 변경됩니다. Chrome / Edge에서만 지원됩니다.
다운로드 방식: 파일을 드래그 & 드롭하면 변환된 파일명으로 다운로드됩니다.
모든 브라우저에서 사용 가능합니다.
이 브라우저는 로컬 파일 직접 변환을 지원하지 않습니다. 다운로드 방식을 사용해주세요.

macOS는 유니코드 NFD(분해형), Windows는 NFC(조합형)를 사용합니다.

이 차이로 인해 한글 파일명이 깨져 보이는 문제를 해결합니다.

모든 처리는 브라우저에서 이루어지며, 파일이 서버로 전송되지 않습니다.
